# Raw Data

Downloaded source files go here exactly as received from the provider. Keep
provider filenames where possible and record each file in `manifest.csv`.

Suggested subfolders can be created as data are acquired, for example:
`ecostress_lst/`, `ecostress_et/`, `sentinel2/`, `landcover/`, `era5_land/`,
`prism/`, `drought/`, `census/`, `tree_inventory/`, and `building_footprints/`.
